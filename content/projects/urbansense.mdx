# UrbanSense — Full-Stack Geospatial AI Platform for Urban Risk Analysis

- **5-Module ML System at Scale:** Built a geospatial AI platform covering 200+ sq. km with U-Net segmentation, XGBoost/LightGBM risk models, LSTM forecasting, and graph analysis on 50,000+ data points; delivered 20+ REST APIs with interactive visualizations.
- **4-Metric Evaluation Framework:** Established systematic model evaluation using IoU, F1, ROC-AUC, and RMSE — aligned with production ML standards for bias detection, accuracy, and deployment readiness.
