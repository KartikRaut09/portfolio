# CloudEdge — Multi-Agent LLM Simulator for Cloud Crisis Management

- **GRPO RL Fine-tuning Pipeline:** Built a GRPO reinforcement learning system (30 episodes × 512 steps) training LLM agents for autonomous cloud infrastructure decisions; improved reward +48% (4.6 → 6.8) and reduced policy entropy from 0.50 → 0.02, demonstrating end-to-end fine-tuning pipeline mastery.
- **Agentic Multi-Agent Architecture:** Designed a 3-agent negotiation system (resource, cost, sustainability agents) across 7 crisis scenarios with tool use and inter-agent communication; published live on HuggingFace Spaces via Docker.
